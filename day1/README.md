# Assignment 1 – Rule-Based AI Agent using LangChain and Groq

## Overview

This assignment introduces the concept of a **basic AI agent** using rule-based reasoning.
The agent takes user input, identifies the intent using simple keyword matching, and performs an appropriate action.

The system follows the **Agent Pipeline Architecture**:

```
User Input → Intent Detection → Action Execution → Response
```

The agent also uses an **LLM fallback** via the **Groq API (Llama-3.3-70B-Versatile)** when the input does not match any predefined rule.

---

# Features

* Simple **rule-based AI agent**
* Handles multiple user intents
* Supports **basic calculations**
* Returns **current system date**
* Responds to greetings
* Uses **Groq LLM for fallback queries**
* Secure API key management using `.env`

## 4. Actions

Each intent triggers a specific function:

| Intent      | Function      | Description                        |
| ----------- | ------------- | ---------------------------------- |
| Greeting    | `greet()`     | Responds with a greeting           |
| Date        | `get_date()`  | Returns the current date           |
| Calculation | `calculate()` | Evaluates mathematical expressions |
| Fallback    | `ask_llm()`   | Queries Groq LLM                   |

## 5. LLM Integration

If no rule matches, the query is sent to the **Groq LLM**.

